Rename to `grid`. The abbreviation `g` saves 4 characters but forces every reader to mentally map it — especially since the function *returns* it, making the name part of the caller's context. In a 4-line function, clarity > brevity. Also, this code has a **bug**: `[[0] * cols] * rows` creates `rows` references to the *same* inner list. Fix: `[[0] * cols for _ in range(rows)]`. Then name the result `grid`.
