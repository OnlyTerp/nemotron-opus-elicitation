Yes, add them. Even short functions benefit from type hints—they document intent, catch errors early, and help IDEs autocomplete.

```python
from typing import Sequence

def moving_average(xs: Sequence[float], k: int) -> list[float]:
    out = []
    for i in range(len(xs) - k):
        window = xs[i:i+k]
        out.append(sum(window) / k)
    return out
```

Use `Sequence[float]` (or `list[float]` if you only accept lists) for the input so callers can pass tuples, arrays, etc. `k` is `int`. Return type is `list[float]`.
