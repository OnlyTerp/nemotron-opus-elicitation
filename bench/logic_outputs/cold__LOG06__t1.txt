Add a one-line docstring. The logic is simple, but the *contract* isn't: returns `None` if no even number exists, otherwise the first even doubled. That edge case (`None` return) is the exact thing callers will get wrong without a hint. A docstring like `"""Return first even number doubled, or None if none exist."""` costs one line and prevents misuse. Skip docstrings only for truly obvious pure functions (e.g., `add(a, b)`).
