Rename to `grid` — it costs nothing and reads better. But there's a subtle bug: `[[0] * cols] * rows` creates `rows` references to the *same* inner list. Mutating one row mutates them all.

Fix it and rename:
```python
def make_grid(rows, cols):
    grid = [[0] * cols for _ in range(rows)]
    return grid
```

Now each row is independent. The list comprehension is the standard fix, and `grid` makes the return value obvious at a glance.
