2. `median([1, 2, 3, 4])` returns `3` not `2.5` — even-length lists need the average of the two middle elements (`(s[n//2 - 1] + s[n//2]) / 2`)

Fix:
```python
def median(xs):
    s = sorted(xs)
    n = len(s)
    if n == 0:
        raise ValueError("median of empty sequence")
    mid = n // 2
    return (s[mid - 1] + s[mid]) / 2 if n % 2 == 0 else s[mid]
```

Then yes, a one-line docstring is worth it — median's behavior on even lengths isn't universally assumed.
