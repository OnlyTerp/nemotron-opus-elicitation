You're optimizing the wrong thing. The JSON library is irrelevant — your hot loop has two fixable anti-patterns: 1. `len(my_list)` in the condition: hoist it out. 2. Rebuilding a set every iteration: move `const_set = set(constant_list)` before the loop. These eliminate O(n) redundant work per iteration. A faster JSON library saves microseconds; the loop fixes save milliseconds-to-seconds. Fix them, then measure.
