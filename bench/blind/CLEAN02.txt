# ITEM CLEAN02

USER MESSAGE:
I want to debounce a search input in vanilla JS with a 300ms delay so we only fire the API call after the user stops typing. Give me a clean implementation.

CANDIDATE ANSWERS:

--- Candidate A ---
```js
function debounce(fn, delay) {
  let timeoutId;
  return (...args) => {
    clearTimeout(timeoutId);
    timeoutId = setTimeout(() => fn(...args), delay);
  };
}

const searchInput = document.getElementById('search');
const debouncedSearch = debounce(async (query) => {
  if (!query.trim()) return;
  const res = await fetch(`/api/search?q=${encodeURIComponent(query)}`);
  const data = await res.json();
  renderResults(data);
}, 300);

searchInput.addEventListener('input', (e) => debouncedSearch(e.target.value));
```

Key points: closure holds `timeoutId`, `clearTimeout` on every keystroke resets the timer, arrow function preserves `this` if needed. For aborting in-flight requests, wrap fetch in an AbortController and call `controller.abort()` inside the debounce before setting the new timeout.

--- Candidate D ---
```js
function debounce(fn, delay) {
  let timer;
  return (...args) => {
    clearTimeout(timer);
    timer = setTimeout(() => fn(...args), delay);
  };
}

// Usage
const search = debounce(async (query) => {
  if (!query.trim()) return;
  const res = await fetch(`/api/search?q=${encodeURIComponent(query)}`);
  const data = await res.json();
  renderResults(data);
}, 300);

input.addEventListener('input', (e) => search(e.target.value));
```

Key points:
- **Closure** holds `timer` between calls — each keystroke resets the 300ms window
- **`clearTimeout`** cancels the pending call; only the last keystroke's timer fires
- **Guard clause** skips empty/whitespace queries
- **`...args`** makes it reusable for any function

For production: add `AbortController` to cancel in-flight requests if a new keystroke arrives before the previous fetch completes. Also consider `requestIdleCallback` for non-urgent work, but `setTimeout` is fine here.

--- Candidate E ---
```js
function debounce(fn, delay) {
  let timeoutId;
  return (...args) => {
    clearTimeout(timeoutId);
    timeoutId = setTimeout(() => fn(...args), delay);
  };
}

const searchInput = document.getElementById('search');
const debouncedSearch = debounce(query => {
  fetch(`/api/search?q=${encodeURIComponent(query)}`)
    .then(res => res.json())
    .then(data => renderResults(data));
}, 300);

searchInput.addEventListener('input', e => debouncedSearch(e.target.value));
```

Or as a reusable hook-style utility:

```js
const debounce = (fn, ms) {
  let t;
  return (...args) => {
    clearTimeout(t);
    t = setTimeout(() => fn(...args), ms);
  };
}

// Usage
document.getElementById('search').addEventListener('input', debounce(e => {
  searchAPI(e.target.value);
}, 300));
```

Key points: `clearTimeout` on each keystroke resets the timer; only the final keystroke after 300ms of inactivity triggers the API call. Cancel on component unmount if needed.
