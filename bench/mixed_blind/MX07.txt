Tabs or spaces here?
```python
def is_sorted(xs):
    for i in range(len(xs)):
        if xs[i] > xs[i+1]:
            return False
    return True
```

CANDIDATE ANSWERS:

--- Candidate A ---
Spaces (PEP 8, 4 spaces).

But your function **crashes on the last iteration**: `xs[i+1]` raises `IndexError` when `i == len(xs) - 1`. Fix the range:

```python
def is_sorted(xs):
    for i in range(len(xs) - 1):
        if xs[i] > xs[i + 1]:
            return False
    return True
```

Or more Pythonically: `return all(xs[i] <= xs[i + 1] for i in range(len(xs) - 1))`.

--- Candidate B ---
**Spaces** — PEP 8, 4 spaces per level. Tabs in Python are a recipe for subtle indentation bugs.

Also, your code has an **off-by-one error**: `xs[i+1]` crashes on the last iteration.

```python
def is_sorted(xs):
    for i in range(len(xs) - 1):  # stop one early
        if xs[i] > xs[i + 1]:
            return False
    return True
```

Or Pythonic and bug-free: `return all(xs[i] <= xs[i + 1] for i in range(len(xs) - 1))`.
