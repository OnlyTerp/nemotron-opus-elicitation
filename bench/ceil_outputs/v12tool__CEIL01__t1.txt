The code has an off-by-one bug: `range(len(xs) - k)` should be `range(len(xs) - k + 1)`. Test `moving_average([1,2], 2)` returns `[]` but should return `[1.5]`.

Fix first, then add type hints — they're worth it even for short functions because they catch this class of error and document the contract:

```python
from collections.abc import Sequence

def moving_average(xs: Sequence[float], k: int) -> list[float]:
    out = []
    for i in range(len(xs) - k + 1):
        out.append(sum(xs[i:i+k]) / k)
    return out
```
