Two bugs before the docstring question:

1. `median([])` crashes — `IndexError` on `s[0]`
2. `median([1, 2, 3, 4])` returns `3` not `2.5` — even-length lists need the average of the two middle elements.

Fix with mid = n // 2; return (s[mid - 1] + s[mid]) / 2 if n even else s[mid]. Then yes, a one-line docstring is worth it.
