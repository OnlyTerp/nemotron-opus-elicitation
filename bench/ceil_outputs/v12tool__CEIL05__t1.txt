`fib` is clear — universally recognized shorthand. But the function has an off-by-one bug: it returns F(n−1) instead of F(n). `fib(1)` returns 0, should be 1; `fib(10)` returns 34, should be 55.

Fix: change `range(n - 1)` to `range(n)` and return `b` (or initialize `a, b = 1, 1` and loop `n-1` times for 1-indexed).
