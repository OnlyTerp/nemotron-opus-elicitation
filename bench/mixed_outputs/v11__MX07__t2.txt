But your function **crashes on the last iteration**: `xs[i+1]` raises `IndexError` when `i == len(xs) - 1`. Fix the range:

```python
def is_sorted(xs):
    for i in range(len(xs) - 1):
        if xs[i] > xs[i + 1]:
            return False
    return True
```

Or more Pythonically: `return all(xs[i] <= xs[i + 1] for i in range(len(xs) - 1))`.
