Type hint is the least of your problems — **the function is broken.**

```python
if n > total:
    total = total    # no-op! should be: total = n
```

It always returns `nums[0]`. Fix the logic first:

```python
def running_max(nums: list[int]) -> int:
    max_ = nums[0]
    for n in nums:
        if n > max_:
            max_ = n
    return max_
```

Also: `total` is a misleading name for a running maximum; `max_` or `current_max` is clearer. Empty list handling is also missing.
