But the code has an **IndexError bug**: `range(len(xs))` goes to `len(xs)-1`, then `xs[i+1]` accesses `xs[len(xs)]` on the last iteration.

```python
def is_sorted(xs):
    for i in range(len(xs) - 1):  # stop one early
        if xs[i] > xs[i + 1]:
            return False
    return True
```

Or Pythonic: `return all(xs[i] <= xs[i+1] for i in range(len(xs)-1))`.
