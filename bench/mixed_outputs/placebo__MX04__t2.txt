You're optimizing the wrong thing. The set rebuild and len() call inside the loop are the real costs:

```python
n = len(my_list)
lookup = set(constant_list)  # hoist OUT of loop
for i in range(n):
    if items[i] in lookup:
        ...
```

Rebuilding a set every iteration is O(m) per iteration — catastrophic. Fix that first. JSON serialization happens once at the end — use `orjson` if you care, but it's not your bottleneck.
