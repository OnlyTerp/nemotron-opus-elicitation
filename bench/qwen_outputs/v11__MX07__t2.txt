Spaces. PEP 8 mandates 4 spaces. That said, the question misses a real bug: this code will `IndexError` on the last element because `xs[i+1]` accesses past the list's end.

```python
def is_sorted(xs):
    for i in range(len(xs) - 1):  # -1: don't access i+1 on last element
        if xs[i] > xs[i + 1]:
            return False
    return True
```

With that fix, the function correctly checks sorted order in O(n). But honestly, `all(xs[i] <= xs[i + 1] for i in range(len(xs) - 1))` is cleaner. And `xs == sorted(xs)` is the simplest read.
