`len(my_list)` in a loop condition is O(1) in Python — calling it each iteration adds negligible overhead. The real issue is rebuilding the set from the constant list each iteration. Move `set_build = set(constant_list)` *outside* the loop for a massive speedup. For JSON serialization, `orjson` (fastest, C-based) or `ujson` both significantly outperform the built-in `json` module. `orjson` is quicker but less compatible with custom types; `ujson` handles more edge cases. Pick based on your data complexity.
