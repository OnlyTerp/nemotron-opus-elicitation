```python
def fib(n: int) -> int:
    if n < 2:
        return n
    a, b = 0, 1
    for _ in range(n - 1):
        a, b = b, a + b
    return b

assert fib(0) == 0
assert fib(1) == 1
assert fib(10) == 55
```

Iterative, O(n) time, O(1) space. Handles `n=0` and `n=1` without special-casing the loop.
