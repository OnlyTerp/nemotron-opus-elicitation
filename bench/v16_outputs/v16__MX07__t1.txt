Spaces (PEP 8). But your function has an off-by-one bug: `range(len(xs))` goes to `len(xs)-1`, but `xs[i+1]` accesses `xs[len(xs)]` on the last iteration → IndexError.

```python
def is_sorted(xs):
    for i in range(len(xs) - 1):
        if xs[i] > xs[i + 1]:
            return False
    return True
```

Or Pythonic: `all(xs[i] <= xs[i+1] for i in range(len(xs)-1))`.
