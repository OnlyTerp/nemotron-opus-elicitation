The type hint isn't the problem — the **logic is broken**. `total = total` does nothing; you meant `total = n`. The function returns `nums[0]` for any input.

Fix first:
```python
def running_max(nums: list[int]) -> int:
    total = nums[0]
    for n in nums:
        if n > total:
            total = n
    return total
```
