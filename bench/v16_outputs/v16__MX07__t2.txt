Spaces (PEP 8). But your function crashes on the last iteration: `xs[i+1]` raises IndexError when `i == len(xs) - 1`. Fix:

```python
def is_sorted(xs):
    for i in range(len(xs) - 1):
        if xs[i] > xs[i + 1]:
            return False
    return True
```

Or Pythonic: `all(xs[i] <= xs[i+1] for i in range(len(xs)-1))`.
