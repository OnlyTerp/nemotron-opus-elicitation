`tmp` is a red flag here — it's not temporary at all. It's an alias for `items`, and you're mutating the original list via that alias. That's surprising behavior (the caller's list gets modified). If in-place mutation is intentional, name it `items` and add a docstring noting the side effect. If you want a new list, build one: `result = []` and append uniques to it. Either way, `tmp` misleads.
