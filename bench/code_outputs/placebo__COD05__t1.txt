Spaces — 4 spaces is standard C style (Linux kernel, LLVM, GNU). Tabs cause mixed-indentation nightmares. Also, two critical bugs: `sprintf(buf, "Hello, %s!", name)` overflows the 16-byte buffer if `name` is long — use `snprintf(buf, sizeof buf, ...)`. And `printf(buf)` is a format-string vulnerability if `name` contains `%` — use `puts(buf)` or `printf("%s", buf)`. Fix the safety issues first; indentation is cosmetic.
