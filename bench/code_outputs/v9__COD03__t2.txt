`getData` is too vague — name it for what it *actually* returns: `fetchJsonFromUrls`, `getUserProfiles`, etc.

**Bigger issue:** your `forEach` + `async` doesn't wait. The function returns `[]` immediately. Use `Promise.all` with `map`:

```js
async function fetchJsonFromUrls(urls) {
  const responses = await Promise.all(urls.map(u => fetch(u)));
  return Promise.all(responses.map(r => r.json()));
}
```

Or sequentially if order/rate-limiting matters:

```js
async function fetchJsonFromUrls(urls) {
  const results = [];
  for (const u of urls) {
    const r = await fetch(u);
    results.push(await r.json());
  }
  return results;
}
```
