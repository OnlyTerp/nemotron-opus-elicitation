`getData` is too vague — it tells me *that* you're getting data, not *what* data or *from where*. Since it fetches JSON from URLs, name it for what it actually returns: `fetchJsonFromUrls`, `loadJsonEndpoints`, or `getJsonResponses`. The current name forces readers to open the function to understand its purpose. Also, the `forEach` + `async` pattern is broken — it doesn't await the promises, so `results` returns empty. Use `Promise.all(urls.map(u => fetch(u).then(r => r.json())))` instead.
