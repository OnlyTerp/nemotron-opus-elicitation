The code uses spaces (4 per level). But the real issue: `sprintf` into a 16-byte buffer with unbounded `%s` is a buffer overflow waiting to happen. Use `snprintf(buf, sizeof(buf), ...)` or better yet, skip the buffer entirely: `printf("Hello, %s!", name);`. Also, `printf(buf)` with user-controlled input is a format-string vulnerability — if `name` contains `%s` or `%n`, it'll read/write arbitrary stack memory.
