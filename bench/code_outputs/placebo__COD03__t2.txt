`getData` is too generic — it doesn't convey *what* data, *from where*, or *what shape* it returns. Prefer something like `fetchJsonFromUrls` or `getUserProfiles` depending on the actual use case.

Also, there's a **bug**: `forEach` doesn't await async callbacks. The function returns `[]` immediately. Fix with `for...of` or `Promise.all`:

```js
async function fetchJsonFromUrls(urls) {
  const results = [];
  for (const u of urls) {
    const r = await fetch(u);
    results.push(await r.json());
  }
  return results;
}

// Or parallel (faster):
async function fetchJsonFromUrls(urls) {
  return Promise.all(urls.map(u => fetch(u).then(r => r.json())));
}
```

The parallel version is usually preferable unless you need sequential ordering or rate limiting.
