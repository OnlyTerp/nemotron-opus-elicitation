1024 is too small — you're making a syscall per KB. Use **64KB–1MB** (65536–1048576). 64KB is a sweet spot: large enough to amortize syscall overhead, small enough to stay in L2 cache and not pressure memory.

Better yet, use `shutil.copyfileobj` which picks a good default (16KB in CPython, but internally uses `sendfile`/`copy_file_range` on Linux where available):

```python
import shutil

def copy_file(src, dst):
    with open(src, 'rb') as f, open(dst, 'wb') as g:
        shutil.copyfileobj(f, g)  # uses 16KB default; kernel handles rest
```

If you need manual control, 65536 (64KB) is a solid constant.
