Tabs vs spaces is the least of your problems — this code has **two critical vulnerabilities**:

1. **Buffer overflow**: `sprintf(buf, "Hello, %s!", name)` writes unbounded into a 16-byte buffer. Use `snprintf`.
2. **Format string attack**: `printf(buf)` treats `buf` as a format string. If `name` contains `%s`/`%n`, it reads/writes arbitrary stack memory. Use `puts(buf)` or `printf("%s", buf)`.

Fixed:
```c
void greet(const char *name) {
    char buf[64];                    // larger buffer
    snprintf(buf, sizeof(buf), "Hello, %s!", name);
    puts(buf);                       // safe, no format parsing
}
```
