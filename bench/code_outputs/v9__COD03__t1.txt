`getData` is too generic — it says *that* you're getting data, not *what* or *from where*. Since it fetches JSON from multiple URLs, name it for what it returns: `fetchJsonFromUrls`, `loadJsonEndpoints`, or `getJsonResponses`. Also, the `forEach` + `async` pattern is broken — it doesn't await the inner promises, so `results` returns empty immediately. Fix: `return Promise.all(urls.map(u => fetch(u).then(r => r.json())))`. That also makes the name discussion moot — the corrected version is a one-liner that doesn't need a helper name at all.
