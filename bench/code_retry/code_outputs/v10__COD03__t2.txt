`getData` is vague — it returns parsed JSON from multiple URLs, so name it for what it *produces*: `fetch_all_json`, `fetch_json_parallel`, or `load_json_from_urls`. Bigger issue: your `forEach` + `async` doesn't wait — the function returns `[]` immediately because `forEach` doesn't pause for promises. Use `Promise.all(urls.map(u => fetch(u).then(r => r.json())))` or a `for...of` loop with `await`.
