That snippet has a buffer overflow (`sprintf` into `buf[16]` with unbounded `name`) and a format-string vulnerability (`printf(buf)`). Fix those first — tabs vs spaces is the least of its problems.

```c
void greet(const char *name) {
    char buf[64];
    snprintf(buf, sizeof(buf), "Hello, %s!", name);
    puts(buf);
}
```
